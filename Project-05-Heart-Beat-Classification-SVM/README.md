# Heart Beat Classification using Support Vector Machine (SVM)

## Goal
Build a machine learning model to classify heart beat signals, introducing classical classification techniques for biomedical signal data.

## Approach
- Loaded and explored heart beat signal data
- Performed feature preprocessing and label preparation
- Trained a Support Vector Machine (SVM) classifier
- Evaluated model performance using standard classification metrics

## Key Learnings
- Applying classical ML models to biomedical signal classification
- Understanding the strengths of SVMs for structured feature spaces
- Evaluating classification performance on health-related datasets

## Tools Used
- Python
- NumPy, Pandas
- scikit-learn
- Matplotlib
