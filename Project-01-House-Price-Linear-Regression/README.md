# House Price Prediction using Linear Regression

## Goal
Build a linear regression model to predict house prices based on input features, as an introduction to supervised learning and regression analysis.

## Approach
- Performed basic data preprocessing and feature selection
- Trained a Linear Regression model
- Evaluated model performance using standard regression metrics

## Key Learnings
- Understanding the relationship between features and target variables
- Interpreting linear regression coefficients
- Evaluating regression models and identifying underfitting or overfitting

## Tools Used
- Python
- NumPy, Pandas
- scikit-learn
- Matplotlib
